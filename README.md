## What we built

Algorithm Odyssey is a web-based interactive learning platform for DAA. The idea was simple — algorithms are hard to understand just by reading. So we built something where you actually *do* things: move disks, sort bars, watch trees split, race algorithms against each other.

The project currently has a frontend built in HTML/CSS/JavaScript and C++ algorithm implementations. The next phase connects the two — a C++ backend that serves real algorithm data, timing, and step output directly to the website.

---

## How a module works

Every chapter follows the same structure so it feels consistent no matter which topic you're on:

**Learn** — the concept is explained with code snippets and complexity tables. No walls of text, just what you need to understand the idea.

**Watch** — handpicked YouTube videos for each topic. Watch before you visualize — it clicks faster.

**Visualize** — see the algorithm running step by step. You control the input and speed, and every operation is shown clearly so you can follow along.

**Compare** — run all algorithms on the same input and see the difference in time and operations. This is where you actually see why O(n log n) beats O(n²).

**Game** — you play the algorithm. In sorting you manually swap bars. In recursion you solve Tower of Hanoi. The game makes you think like the algorithm, not just watch it.

Students earn **XP** for each section they complete and unlock **badges** as they finish chapters. Progress is saved in the browser so it carries over between sessions.
